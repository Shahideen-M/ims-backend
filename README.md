Spring Boot backend for managing users and products.  
Secure with JWT authentication, users can register, log in, manage their profile, and perform CRUD operations on products.

## Features
- User registration, login, profile management, and deletion
- Product CRUD operations
- JWT-based security

## Tech Stack
- Java + Spring Boot
- Spring Security (JWT)
- MySQL
- Maven

## Running Locally
1. Clone the repo:
   git clone https://github.com/Shahideen-M/ims-backend.git and 
   Configure MySQL in application.properties and your secret key

3. Build & run:
- Copy code
- mvn clean install 
- mvn spring-boot:run
